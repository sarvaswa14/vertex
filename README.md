# Vertex

Built this as a productivity app to organize tasks, track focus sessions and take notes.

## What it does
- Add, complete and delete tasks
- Pomodoro timer that saves your focus sessions
- Notes with create, edit and delete
- Analytics showing your productivity stats
- Dark and light mode
- Login and signup with JWT auth

## Stack
MongoDB, Express, React, Node.js
